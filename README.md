Projeto e Análise de Algoritmos

Professor: Danilo Medeiros Eller

Trabalho 1: análise de algoritmos de ordenação

- BubbleSort, FlaggedBubbleSort, HeapSort, InsertionSort, MergeSort, QuickSort (pivo inicial e central), SelectionSort, ShellSort
- Comparação visual dos tempos de ordenação

Trabalho 2: algoritmos e métodos de implementação
- Associação de tarefas (Assignment Problem) com: tentativa e erro com Branch and Bound
- Codificação de Huffman com: algoritmo guloso
- Mochila Fracionária (Fractional Knapsack Problem) com: algoritmo guloso
- Mochila Booleana (Knapsack Problem) com: Programação Dinâmica
- Subsequência Comum Máxima (Longest Common Subsequence) com: Programação Dinâmica
